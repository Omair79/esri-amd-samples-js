ArcGIS for JavaScript API with Dojo Bootstrap

About

This is some boilerplate code for working with Esri's ArcGIS for JavaScript API and Dojo Bootstrap (two great tastes that taste great together). If you see tabs and a map, it's working!

Add Dojo Bootstrap, Dojo, and/or ArcGIS for JavaScript API modules as needed to test that they play nicely together.
